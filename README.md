# RecipeBase &middot; [![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)


## Description 
<p>RecipeBase is a simple base library for recipe management using EF Core in C#.</p>
<p>This base is used in multiple HMI applications for dosing machines.</p>

**<p>This library is just a simple base.<br/>Properties, etc. must be customized according to your use case.</p>**
